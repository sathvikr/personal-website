I’m launching Instachip at Stanford, with Jag Maddipatla. Instachip is a Copilot for FPGA programming, except its code suggestions are always right, because we use a better algorithm.

Previously, I designed semiconductor chips for ~3 years. Jag and I built Tachyon, a chip for O(1) matrix-vector multiplication. Also, we created a concept for an Optical Vector Processor, which is like a GPU but with silicon photonics (US18119437, pending).

I've also built programming languages, compilers, and interpreters for ~6 years. My favorite is Apollo, which

  • is a tensor algebra DSL
  • compiles to an optical processor
  • stores sparse tensors better than NumPy

Before Apollo, I implemented a regex interpreter using Brzozowski derivatives, a two-stage compiler for the Jack language (following this textbook), and an interpreter for RED, my first programming language, which I made when I was 12.
