Taylor Swift's implementation of the BogoSort algorithm

"The process of preparing programs for a digital computer is especially attractive, not only because it can be economically and scientifically rewarding, but also because it can be an aesthetic experience much like composing poetry or music." I also believe in the converse: creative formalization of music into algorithms is a unique representation of what computer science embodies.

So, for fun, I invented a poetic variation of the "Swift" programming language: Taylor-Swift. And I used this new language to code the famous BogoSort algorithm (shuffle a list of numbers randomly; check if sorted; if not, repeat the process, until a sorted list appears).
