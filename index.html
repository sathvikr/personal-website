I’m launching Instachip with Jag Maddipatla. Instachip is a tool that lets you "pip install" chip components.

If you're into chips, FPGAs, or being a miscellaneous hardware geek, check out OpenChips! It's an open source repository for chip design. Chips are becoming very important and this repository could play an interesting role...

Previously, I designed semiconductor chips for ~3 years. Jag and I built Tachyon, a chip for O(1) matrix-vector multiplication, and designed the Vector Processing Unit (US18119437, pending).

I also designed and implemented a programming language for optical computers. Specifically, it is a tensor algebra DSL that compiles to an optical processor and stores sparse tensors better than NumPy. Before this "optical language" (wow!), I implemented a regex interpreter using Brzozowski derivatives, a compiler for the Jack programming language (following this textbook), and an interpreter for RED, my first programming language, which I made when I was 12.
